# 프로젝트 스타일 가이드

- **디자인 컨셉**: '한국적인 현대미'를 추구함. 차분한 베이지, 브라운 톤(#f5f0e8, #5c4433)을 주로 사용함.
- **기술 스택**: React, Tailwind CSS, Framer Motion 사용.
- **레이아웃**: 모든 섹션은 `w-screen`, `h-screen`을 활용한 풀스크린 구성을 선호함.

# 코딩 규칙

- **이미지 관리**: 모든 이미지는 `src/assets/images/` 폴더에 넣고, 반드시 상단에서 `import`하여 사용함.
- **애니메이션**: `motion/react`(Framer Motion)를 사용하여 부드러운 등장 효과를 기본으로 적용함.
- **반응형**: 모바일에서도 텍스트가 깨지지 않도록 `text-4xl`에서 `text-7xl`까지 유연하게 대응함.
- **코드 스타일**: 가급적 함수형 컴포넌트를 사용하고, Tailwind 클래스는 가독성을 위해 논리적인 순서로 배치함.

# 추가 요청 사항

- 모든 설명과 주석은 **한국어**로 작성해줘.
- 어려운 기술 용어보다는 이해하기 쉬운 일상어로 설명해줘.
